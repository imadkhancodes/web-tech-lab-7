# web-tech-lab-7

### Task 1: Implement Interface in Class

In this task we created an **Employee interface** with properties id, name and salary and a method displayInfo(). Then we created a class **EmployeeDetails** that implements this interface. A constructor was used to set values and the displayInfo method was used to print all employee details on the screen.

### Task 2: Calculate Area of Rectangle

In this task we used an **arrow function** to calculate the area of a rectangle. The function takes two parameters **length and width** of type number and returns the result of length multiplied by width. We called the function with values 10 and 5 and the output was **Area of Rectangle is: 50**.

### Task 3: Interface Assertion

In this task we used **Type Assertion** to assign properties to an empty object following an interface structure. We created a **Student interface** with properties id, name and grade. Then we created an empty object using **as Student** keyword and assigned values to each property one by one and printed the result on the screen.
